Kami mendapat alert tentang 847 kegagalan login SSH dalam satu jam terakhir di ssh.domain.com. Apakah ini serangan aktif?
Ringkasan
Ya — pola ini konsisten dengan serangan brute force SSH credential-stuffing aktif. 847 kegagalan dalam 60 menit dari 3 IP unik melampaui ambang batas klasifikasi serangan otomatis.
Klasifikasi Ancaman
T1110.001 — Brute Force: Password Guessing (MITRE ATT&CK)  ·  TINGGI
Bukti
IP Sumber: 45.152.67.23, 185.220.101.9, 91.240.118.172 — ditandai di AbuseIPDB (>85%). Serangan mulai 14:07 WIB · ~14 percobaan/menit.
Rekomendasi
1. Blokir 3 IP sumber di firewall segera. 2. Nonaktifkan autentikasi kata sandi. 3. Batasi port 22 ke VPN saja. 4. Tinjau log auth untuk login berhasil.
